# Netflix Data Analysis

This project explores Netflix data using Python and Jupyter Notebook.

## Contents
- `notebooks/` → Jupyter notebooks with analysis
- `images/` → Charts and visuals
- `docs/` → Documentation
- `data/` → Dataset files (e.g., `mymoviedb.csv`)

## Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Visualization of trends

## Dataset
The analysis uses `mymoviedb.csv`, located in the `data/` folder.
